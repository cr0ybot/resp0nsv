    Info

    What is resp0nsv?

    Resp0nsv is a tool for web developers creating responsive websites.

    At it's core, it is a manipulable iframe, within which is the page you initialized resp0nsv on. This is not ideal, because it makes inspection with developer tools a bit cumbersome, but it is perhaps easier to use for simple viewing than Chrome's built-in "device-metrics overrides". If you couldn't tell, resp0nsv was inspired by the built-in Firefox "Responsive Design View".

    Why should I use resp0nsv? It's just a stupid iframe.

    It's more than just an iframe. You can save your own screen size presets, resize it by hand without your browser chrome getting in the way, and you can even "rotate" the screen.

    If you aren't convinced, or if you've got a suggestion that would make this more useful, please use the "Send Feedback" link on the Chrome Extension web store page.

    How do I use it?

    You can change the size of the responsive preview in a number of ways:

    1. Use the resizing handles on the right, bottom, and bottom-right corner.
    2. Type in the width and height you want at the upper-left, under the url. You can also activate the input by clicking, and use your arrow keys or mouse wheel to change the number incrementally.
    3. Choose a size preset from the preset list. These presets can be modified in the Size Presets pane.

    Why is the background transparent?

    You didn't set a background-color on your body or html element.
